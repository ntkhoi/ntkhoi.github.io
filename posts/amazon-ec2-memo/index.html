Amazon EC2 is AWS primary web service that provides resizable compute capacity in the cloud.
Amazon EC2 is allows you to acquire compute through the lanching of viturak servers call instances.

Instance Types

Instance Types defines the vitural hardware supporting an Amazon EC2 instance.
There are many instance types available follow these dimension.

  • vCPU
  • Memory
  • Storage (Size/Type)
  • Network perormance Instance types are group into families base on the ratio of these values to each other

Amazon Machine Images (AMIs).

THe Amazon machine images defines the initial software the will be on an instance when it is launched. AMIs contain:

  • The OS and it’s configuration
  • The initial state of any patches
  • Application or system software There are 4 source of AMIs.
  • Pushlished by AWS
  • The AWS marketplace
  • Generated from an existing instances.
  • Uploaded virtual server

Addressing an Instance

  • Public Domain Name System (DNS): Name created when launch an instance.
  • Public ID : Also created wheb launch an instance.
  • Elastic IP: is an IP address associate with EC2 instance, persits untill customer release and can be transfer.
